image文件
```sh
# 列出本机的所有 image 文件。
$ docker image ls

# 删除 image 文件
$ docker image rm [imageName]

# 拉取image
$ docker image pull library/hello-world
$ docker image pull hello-world

# 查看本机的image
$ docker image ls

# 运行image
$ docker container run hello-word

```
