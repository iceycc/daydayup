# 目录说明

```
|-- apps 示例
  +-- ... 课程随堂代码 
  +-- ... 每个目录是一个独立的示例
|-- lib 基础库
  +-- model 模型相关
  +-- shapes 各种形状
  +-- tests lib的测试
  +-- game 游戏相关
  +-- time 时间控制
  +-- widget 前端组件（非webgl写的，一般是React)
|-- static 静态资源

```

# 使用方法

```
# 到有package.json的目录(cg的根目录)
# 安装依赖
# 先安装node.js
npm install

# 执行程序
npm start
```
